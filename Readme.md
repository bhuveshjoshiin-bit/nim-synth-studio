# 🚀 Nim Synth Studio

**«A next-generation Web IDE + AI coding platform powered by NVIDIA NIM ⚡
Build, edit, preview, and deploy projects directly in your browser.»**



### 🌐 Live Demo

**👉 https://nim-synth-studio.lovable.app/**



# ✨ Features

### 🧠 AI Coding Engine
**
- Powered by NVIDIA NIM (no OpenAI / Anthropic)
- Generate full projects, functions, and components
- Smart autocomplete and code assistance
- Multi-language support (JS, TS, Python, C#, C++)

### 💻 Web IDE (Replit-style)

- Full in-browser development environment
- File explorer + code editor + terminal
- Real-time editing and execution

### 👀 Live Preview

- Instant preview panel
- Hot reload experience
- Frontend + backend testing support

🚀 One-Click Deploy

- Publish projects instantly
- Shareable live URLs
- Fast build and deployment system

### 🔗 GitHub Integration

- Import repositories
- Commit and push changes
- Export projects easily

### ⚡ Inngest Integration

- Background jobs and workflows
- Async event processing
- Scalable task handling



# 🏗️ Tech Stack

- Frontend: React / Next.js / Tailwind CSS
- Backend: Node.js / Edge Functions
- AI Engine: NVIDIA NIM API
- Execution: Modal / Serverless Compute
- Automation: Inngest
- Storage: Cloud-based filesystem



# 📦 Installation

```git clone https://github.com/bhuveshjoshiin-bit/nim-synth-studio.git```

```cd nim-synth-studio```

```npm install```

```npm run dev```



# ⚙️ Environment Variables

Create a ".env" file:

```NIM_API_KEY=your_nvidia_nim_key```

```MODAL_API_KEY=your_modal_key```

```INNGEST_API_KEY=your_inngest_key```



# 🧪 Usage
